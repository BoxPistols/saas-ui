# About Storybook

Storybook は UI コンポーネントの開発とテストを効率化するための強力なツールです

- Storybook は、UI コンポーネントの検証やデバッグ、デザインシステムの構築を容易に行えるツールであり、Mui などのテーマとの連携も可能です。
- Storybook 内でドキュメンテーションを作成し、CSF やプレイグラウンドを使用してコンポーネントの動作をテストすることができます。
- Storybook のアドオンを使用することで、コンポーネントのテストやデバッグをより効率的に行うことができます。
